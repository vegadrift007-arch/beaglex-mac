```
很多人觉得 Mac 不会中毒
事实是 macOS 上的恶意软件越来越多 🚨
Shlayer / Adload / XCSSET 这些家族都很活跃

但市面上的 Mac 杀毒软件大都：
· 收费贵 💸
· 联网上传数据 📡
· 后台跑一堆服务 🐌
· 闭源不知道在干嘛 🕳️

干脆自己写了一个开源版 ↓↓↓

🐕 BeagleX —— 魔丸比格，撕碎一切
SwiftUI 原生 · 完全离线 · 代码全公开

—— 它能扫什么 ——

🦠 已知恶意软件
内置 20+ 个 Mac 恶意软件家族特征库
（Shlayer / Adload / Pirrit / KeRanger / XCSSET 等）
扫到一键移到废纸篓

🔑 可疑权限滥用
哪些 app 同时拿了"完全磁盘 + 麦克风 + 摄像头"
但没有开发者签名 → 直接标红
（这种最可能是间谍软件）

🚀 未签名启动项
LaunchAgents / LaunchDaemons 里悄悄运行的二进制
来源不明的全部揪出来

哪些进程在外连，但二进制是未签名的

🧠 实时内存监控
```

## 💡 品牌定位

**产品名：** BeagleX 🐕
**Slogan：** 魔丸比格，撕碎一切 · Hunt malware. Shred junk. Set your Mac free.

**为什么是比格犬：**
- Beagle 是著名的嗅探猎犬 — 嗅出 Mac 上的恶意软件
- "魔丸" 体现破坏力 — 撕碎垃圾文件、撕碎缓存、撕碎恶意软件
- 形象可爱反差萌 — 一只戴墨镜 + 用 MacBook 的比格

**为什么这个角度（安全扫描）：**
- "Mac 清理"赛道已被 Pearcleaner(13k⭐) / Tencent 柠檬(6k⭐) 占满
- 开源 macOS "杀毒/安全扫描"几乎没人做，是空白市场
- 戳到"我的 Mac 安全吗"这个长期被忽视的焦虑
